Dashboard

We build this project with goodwill and a lot of sweat. Starting from the ground, we implemented an intuitive, robust and elegant app for our beloved kids.
Then, we decided to release the app to general audience to share the educational material with the community.
We will like to thank the dev team {MN, HP, FR} as well as benovolant testers. MN will like to dedicate wiJa to his wife for the Amazing support to the project and the sacrifice of sparetimes with him.

WiJa is designed like a simplified Android Phone GUI.
The concept here is to build an environnement where kids can focus on one single task at a time. The UI channels the child into one activity with one main task and nothing else to do. This learning method is proven and widely used in schools.
The progress of the child is assessed according to the accomplissement of the main task. Moving one step at a time, your kid is learning vocabulary basics for his future life.
Others apps are multitasking in order to trap kids in the GUI. Thus, their educational performance cannot be assessed.

Following the concept of single tasking the learning content, we overrided firstly the Android default keyboard. We designed a customized keyboard with the minimal number of keys for activities.
Then, we equiped wiJa's App with an highly responsive keyboard to fit its educational purpose. It improves user's learning experience while avoiding the use of the overcomplex multilayered default Android's keyboard.

WiJa features an Artificial Intelligence named A.L.I. (Automated Language Interpretor) to improve the User Experience. An interface to this Artificial Intelligence is embeded within the SMS activity.
A.L.I. is assessing the child vocabulary and suggest work sessions accordyling. But, A.L.I. is designed to do much more ...
A.L.I. can actually understand human language and decide how to respond accordingly. A.L.I. is implementing advanced mathematical model to parse human language. We will publish the model and licence its usage, when funds to substain the project will be available.
